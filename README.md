Lecture2Go Plugins SDK
======================

This software is based on the Liferay 6.2 http://www.liferay.com

[Lecture2Go](http://lecture2go.uni-hamburg.de/open-source) is an open source platform for media management and distribution. Our goal is to support the free access to knowledge. This software solution is aimed at academic institutions and has to strengthen the blended learning.
